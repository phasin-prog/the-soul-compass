# The Soul's Compass design language

## Character

Contemplative, rigorous, intimate. The interface should feel like a quiet midnight reading room: serious without academic theatre, warm without sentimentality, distinctive without distracting from the text.

## Visual system

- Background: near-black violet neutral.
- Surfaces: one or two restrained elevation steps.
- Accent: warm amber for actions, focus, and provenance.
- Typography: Pridi display, Anuphan body and UI.
- Radius: small to medium; reserve pills for compact metadata.
- Dividers: use to organize real editorial structure, not as decoration.
- Motion: subtle state feedback only; honor reduced motion.

## Public archive

- Lead with a clear reading path or research question.
- Keep article titles and source context dominant.
- Prefer indexes and bibliographic rows for large taxonomies.
- Use one lead story plus secondary stories instead of equal-card grids when hierarchy matters.
- Show source level and internal TPDT status plainly.

## Studio

- Optimize for writing, review, status, and publishing.
- Keep primary action obvious and destructive actions separated.
- Preserve drafts and warn before data loss.
- Use dense layouts only where they improve editorial scanning.

## Avoid

- Generic three-card marketing sections.
- Excessive centered copy.
- Gradient blobs, glass panels, neon borders, and ornamental charts.
- Tiny navigation text or controls below 44px.
- Mixed type systems, arbitrary colors, and page-specific token overrides.
